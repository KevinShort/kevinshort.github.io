A for loop repeats the same action a number of times.

Here's an example: for every value of the color red from 0 to 255, draw a line with that color (remember 0 is black and 255 is pure red). 

for (var color = 0; color < 256; color++) {
    context.strokeStyle= 'rgb(' + color + ',0, 0)';
    context.beginPath();
    context.moveTo(color, 0);
    context.lineTo(color, canvas.height);
    context.stroke();
}

Here's what this code is doing:

for (var color = 0; color < 256; color++)

First it sets the variable 'color' to be 0. Then it says, "Keep doing this until 'color' is not less than 256." And it also adds 1 to 'color' every time it loops.
