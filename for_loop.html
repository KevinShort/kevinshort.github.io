A for loop repeats the same action a number of times.

Take example number 1, right here. Here's the javascript code:

var div = document.getElementById('textDiv');
for (var y = 0; y < 5; y++) {
    div.innerHTML += 'hello ';
}

The first line of this code finds an element in the HTML code with an ID of 'textDiv'.

var div = document.getElementById('textDiv');

The second line starts setting up the code for the for loop. First it sets the variable y to be 0. Then it says, "As long as y is less than 5, run this code." y++ tells it to add 1 to y every time it loops. So this code will run 5 times. Each time it loops, it adds 'hello ' to the inside of the div.

for (var y = 0; y < 5; y++) {
    div.innerHTML += 'hello ';
}




Here's another example: for every value of the color green from 0 to 255, draw a line with that color (remember 0 is black and 255 is pure green). 

var canvas = document.getElementById('canvas');
var context = canvas.getContext('2d');
var y = 0;
for (var color = 0; color < 256; color++) {
    context.strokeStyle= 'rgb(0, '+color+', 0)';
    context.beginPath();
    context.moveTo(0, y);
    context.lineTo(canvas.width, y);
    context.lineWidth = 2;
    context.stroke();
    y += 1;
}
